Quotes

> Fusce dictum rutrum arcu, vitae finibus tellus
> accumsan vel.

Or a single line:

> Maecenas cursus, nunc sed elementum lobortis.

Quotes will even auto-wrap to multiple lines when they get too long.

Code Blocks

Use single backticks `To make an inline code block`.

Or use 3 backticks surrounding your block for multi-line. 

```
let me = "Cameron Ingham"
let verb = "Loves"
let school = "DevMountain"
print("\(me) \(verb) \(school)")
```
